# راهنمای استفاده (فارسی)

## وضعیت فعلی پروژه

این ریلیز هنوز **مرورگر اندروید کامل / APK نهایی** نیست و فعلاً شامل هسته‌ی اولیه‌ی AI Assistant است:

- محاسبه ناحیه برش (Crop) از روی انتخاب کاربر
- مدل تنظیمات AI (Provider/Prompt/Capture)
- سریال‌سازی تنظیمات به JSON

این بخش برای اتصال به پروژه‌ی اصلی Reference Browser آماده شده است.

## نحوه اجرا و تست

پیش‌نیاز:
- JDK 21 (ترجیحاً)
- Gradle 8+

فرمان تست ماژول:

```bash
gradle :aiassistant-core:test
```

اگر خطای ناسازگاری Java/Gradle گرفتید، با JDK 21 اجرا کنید:

```bash
JAVA_HOME=<path-to-jdk21> gradle :aiassistant-core:test
```

## خروجی قابل استفاده در این مرحله

در این نسخه، خروجی اصلی یک **ماژول Kotlin قابل ادغام** در اپ اندروید است (نه APK مستقل).

فایل‌های کلیدی:
- `CropMath.kt` برای محاسبه مستطیل برش
- `AiSettings.kt` برای تنظیمات AI Assistant
