# آرشیو درخواست‌ها و نسخه‌های قبلی (فارسی)

این فایل برای نگه‌داری خلاصه‌ی درخواست‌های قبلی شما داخل مخزن است.

## خلاصه درخواست‌های کلیدی

1. ظاهر اپ تا حد ممکن شبیه مرورگر پایه باشد.
2. قابلیت AI فقط برای اسکرین‌شات ناحیه‌ی انتخابی + ارسال به API (خصوصاً ChatGPT/Gemini).
3. تنظیم APIها در Settings با امکان مرتب‌سازی/اولویت.
4. Failover: هر API دو بار با timeout ده‌ثانیه‌ای امتحان شود و سپس API بعدی.
5. Prompt list: امکان ارسال prompt انتخابی یا ارسال بدون prompt.
6. خروجی عدد گزینه برای سوالات تستی به‌صورت واضح و غیرمخفی.

## وضعیت اجرایی فعلی

- سورس اندروید پایه (`app`) اضافه شده است.
- هسته‌ی AI (`aiassistant-core`) برای منطق crop/settings/failover/prompt آماده است.
- برای رسیدن به سطح نهایی GeckoView/Reference Browser، ادغام مرحله بعدی نیاز است.

## نکته درباره حالت مخفی

مطابق سیاست پروژه، قابلیت AI باید شفاف و قابل مشاهده باشد.
بنابراین حالت مخفی/نامرئی/استتاری پیاده‌سازی نشده و نخواهد شد.
