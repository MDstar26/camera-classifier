# آرشیو درخواست‌ها و نسخه‌های قبلی (فارسی)

این فایل برای نگه‌داری خلاصه‌ی درخواست‌های قبلی شما داخل خود مخزن اضافه شده تا در پکیج GitHub نیز موجود باشد.

## خلاصه درخواست‌های کلیدی

1. ظاهر اپ تا حد ممکن شبیه مرورگر پایه باشد.
2. قابلیت AI فقط برای اسکرین‌شات ناحیه‌ی انتخابی + ارسال به API (خصوصاً ChatGPT/Gemini).
3. تنظیم APIها در Settings با امکان مرتب‌سازی/اولویت.
4. Failover: هر API دو بار با timeout ده‌ثانیه‌ای امتحان شود؛ در صورت شکست از چرخه خارج شود و API بعدی تست شود.
5. Prompt list: قبل از عملیات بتوان انتخاب کرد کدام prompt ارسال شود یا هیچ‌کدام ارسال نشود.
6. در سوالات تستی، خروجی عدد گزینه به‌صورت واضح در UI نشان داده شود (غیرمخفی).

## وضعیت اجرایی در این مخزن

- منطق هسته‌ای موردنیاز (settings/failover/prompt/answer extraction/crop math) اضافه شده است.
- این مخزن هنوز شامل سورس کامل اپ اندروید (Reference Browser fork) نیست؛ بنابراین APK نصب‌پذیر هنوز داخل این مخزن تولید نمی‌شود.

## نکته درباره ایده «مخفی/نامرئی»

مطابق سیاست و طراحی این پروژه، قابلیت AI باید شفاف و قابل مشاهده باشد.
بنابراین حالت مخفی/نامرئی/استتاری پیاده‌سازی نشده و نخواهد شد.
