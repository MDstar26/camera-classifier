# وضعیت ساخت APK

## نتیجه بررسی فعلی

در این ریپو ماژول اندرویدی `app` یا سورس کامل مرورگر پایه وجود ندارد.
به همین دلیل APK نصب‌پذیر در این مرحله قابل تولید نیست.

## فایل‌های مرتبط اضافه‌شده

- `scripts/build_android_apk.sh`: اسکریپت کنترل‌شده برای ساخت APK زمانی که سورس کامل اندروید اضافه شد.
- `scripts/create_release.sh`: ساخت آرشیو سورس برای انتشار نسخه فعلی.

## بعد از اضافه شدن سورس اندروید

1. وجود `gradlew` و ماژول `app` را بررسی کنید.
2. دستور زیر را اجرا کنید:

```bash
./scripts/build_android_apk.sh
```

3. فایل خروجی در مسیر `app/build/outputs/apk/debug/*.apk` قرار می‌گیرد.
